---Данные об объявлении:---
0 - объявление заблокировано навсегда
1 - все в порядке. Объявление одобрено, нет подозрений
8 - подозрительное объявление

---Контроллеры:---
+Other - контроллер, отвечающий за "внешние" страницы, теги, лендинг, пользовательское соглашение
+Ads - контроллер отвечающий за отображение обьявлений
+Room - отвечает за добавление, удаление, обновление объявлений и манипуляции с
личным кабинетом
Auth - отвечает за вход, регистрацию, восстановление пароля и т.п
Message - отвечает за сообщения (например, ошибка 404, ajax запрсы и т.п)
Admin - отвчает за функции администрирования
Moder - за функции для модераторов
Api - за API сайта для андроид и т.п устройств
